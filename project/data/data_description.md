# Описание данных

В проекте используется датасет сегментации пневмоторакса на рентгеновских снимках грудной клетки.

## Формат данных

### Изображения
Рентгеновские снимки сохранены в формате `PNG` и лежат в папке:
- `data/png_images/`

### Маски
Бинарные маски пневмоторакса сохранены в формате `PNG` и лежат в папке:
- `data/png_masks/`

### Метаданные
Метаданные и соответствие файлов хранятся в CSV:
- `data/stage_1_train_images.csv`
- `data/stage_1_test_images.csv`

## Поля CSV

В подготовленных CSV могут встречаться поля:
- `new_filename` - имя PNG-файла;
- `ImageId` - исходный идентификатор изображения;
- `has_pneumo` - бинарный индикатор наличия пневмоторакса.

## Как используются данные в проекте

- `train CSV` используется для построения train/val;
- `test CSV` используется как внешний test;
- изображения и маски подаются в `PneumothoraxDataset`;
- метаданные используются для фильтрации, split и EDA.

## Ограничения

В репозиторий включается только небольшой sample-поднабор данных.  
Полный датасет не хранится в git из-за большого размера.
