# Папка `scripts`

В этой папке лежат вспомогательные скрипты, чтобы проект было удобно запускать из консоли.

## Зачем нужна эта папка

Вместо того чтобы каждый раз вспоминать длинные команды, можно использовать готовые скрипты для:
- создания окружения;
- запуска тестов;
- поднятия API;
- smoke-проверки;
- запуска отдельных моделей.

### `setup_env.bat`
Создаёт `.venv` и устанавливает зависимости.

### `run_tests.bat`
Запускает тесты.

### `run_api.bat`
Поднимает FastAPI-сервис.

### `check_project.bat`
Запускает базовую консольную проверку проекта.

## Важно

- `.bat`-скрипты предназначены для Windows CMD / PowerShell;
- в Datasphere `setup_env.bat` не является основным сценарием, потому что `.venv` не переключает текущее Jupyter-ядро.
