# Папка `src`

В этой папке находится основной код проекта.  
Всё, что должно использоваться повторно и запускаться не только из ноутбука, хранится именно здесь.

## Основные части проекта

### `src/train.py`
Точка входа для обучения моделей.  
Запускается через:

```bash
python -m src.train --config configs/training_unet.yaml
```

Именно этот файл:
- читает конфиг;
- строит данные;
- создаёт модель;
- запускает обучение;
- сохраняет артефакты.

### `src/service.py`
FastAPI-сервис для инференса.  
Через него реализованы:
- `/health`
- `/predict`
- `/reload`

Сервис загружает финальную модель, принимает изображение, строит маску и сохраняет результат.

### `src/data/`
Код для работы с данными:
- dataset;
- трансформации;
- вспомогательные функции подготовки входов.

### `src/models/`
Реализации архитектур:
- `UNet`
- `AttentionUNet`
- `TransformerUNet`
- фабрика создания моделей (`factory.py`)

### `src/training/`
Компоненты обучения:
- loss-функции;
- метрики;
- train/eval engine;
- подбор threshold и другие training-вспомогательные части.

### `src/utils/`
Вспомогательный код:
- чтение YAML;
- создание директорий;
- I/O;
- логирование;
- утилиты общего назначения.
