# Папка `notebooks`

В этой папке лежат Jupyter-ноутбуки проекта.  
Они используются не как место для основного кода, а как удобный интерфейс для анализа, демонстрации и проверки.

## Основные ноутбуки

### `exp01_eda_pneumothorax_dataset.ipynb`
Ноутбук для EDA.  
Показывает:
- загрузку данных;
- summary по train/test;
- баланс классов;
- распределение mask coverage;
- примеры изображений и масок;
- базовую проверку корректности подготовки данных.

### `exp02_metrics_model_comparison.ipynb`
Ноутбук для сравнения моделей и артефактов.  
Используется для:
- чтения `history_*.csv`;
- чтения `train_summary_*.json`;
- сравнения `UNet`, `AttentionUNet`, `TransformerUNet`;
- визуального выбора финальной модели по метрикам.

### `service_api_check.ipynb`
Ноутбук для проверки FastAPI-сервиса.  
Через него можно:
- поднять сервис из ноутбука;
- проверить `/health`;
- отправить изображение в `/predict`;
- посмотреть сохранённый `overlay`.

- не хранить в них лишние тяжёлые outputs, если они сильно раздувают репозиторий;
- основной код выносить в `src/`.
