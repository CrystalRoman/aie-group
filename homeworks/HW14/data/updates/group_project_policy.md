# Групповые проекты

По умолчанию итоговый проект выполняется индивидуально. Однако на программах с продуктовым или исследовательским треком координатор может разрешить групповой формат, если группа заранее согласовала тему, роли участников и критерии индивидуального вклада каждого человека.

Даже в групповом проекте каждый студент отвечает за собственный вклад и должен быть готов объяснить решения на защите. Если вклад участника невозможно отделить от вклада команды, комиссия может попросить дополнительные материалы: личный отчёт, журнал задач или индивидуальный созвон.

Без явного согласования групповой формат считать допустимым нельзя. Если студенты подготовили совместный проект самостоятельно, без одобрения команды курса, работа может быть не принята как итоговая аттестация.
