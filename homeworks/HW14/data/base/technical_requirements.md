# Технические требования

Для комфортной работы на курсе нужен ноутбук или стационарный компьютер с современным браузером, стабильным интернетом и правами на установку необходимого программного обеспечения. Желательно иметь не менее 8 ГБ оперативной памяти, чтобы без проблем запускать практические ноутбуки и инструменты для анализа данных.

Студенту понадобятся: актуальная версия Google Chrome или Firefox, архиватор, редактор кода, доступ к терминалу и возможность устанавливать Python-пакеты. Для занятий с видеосвязью нужны наушники и микрофон, а для экзамена или защиты проекта — работающая камера.

Если корпоративный ноутбук ограничивает установку программ, рекомендуется заранее проверить, можно ли использовать личное устройство. Большинство проблем с доступом к платформе можно решить ещё до старта потока, если пройти вводный технический чек-лист.
